\section{Cloud Deployment}
\subsection{Cloud Deployment models}
\begin{itemize}
    \item Private cloud: uses vituralization for on demand resources and can be located off-premises, direct control over resources available, and how data is stored
    \item Public cloud: internet accessible for general public, no access to data center and hardware
    \item Hybrid cloud: mix, for seasonal spikes or cloud bursting
    \item Multicloud : combine different CSP
    \item Community cloud: shared by specific community with common interest
\end{itemize}
